# ByteChat

## About
ByteChat is a full-stack AI chatbot built using the MERN Stack. It allows users to chat with Google's Gemini AI, save conversations, and get programming-related answers through a clean and responsive interface.

## Features
- User Authentication
- AI Chat using Gemini API
- Chat History
- Markdown Support
- Syntax Highlighting
- Copy Code
- Dark/Light Mode
- Responsive Design

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Redux Toolkit
- Tailwind CSS
- JWT Authentication
- Gemini API
